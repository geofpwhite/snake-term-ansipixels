# snake

Simulate the matrix effect in your terminal

## Install
You can get the binary from [releases](https://github.com/fortio/tsnake/releases)

Or just run
```
CGO_ENABLED=0 go install fortio.org/snake@latest  # to install (in ~/go/bin typically) or just
CGO_ENABLED=0 go run fortio.org/snake@latest  # to run without install
```

or
```
brew install ansipixels/tap/snake
```

or
```
docker run -ti fortio/tsnake
```


## Usage

```
snake help

flags:
```
